# NDP电商平台
 - 0.一个简单快捷的电商平台,有用户,商家身份,功能参考京东,苍穹外卖和科协签到网站
 - 基于SpringBoot,SpringMVC,MySql,MybatisPlus开发
 - 1.登录功能已完成,使用Spring Security+Redis+Jwt+Spring mail+RabbitMQ
 - 2.用户,商品,商家,订单功能已完成,使用SpringAOP检查认证及管理缓存
 - 3.收藏,收货地址功能完成(涉及Sql多表查询)
 - 4.数据导出已完成,使用Apache POI实现
 - 5.定时清理任务已完成,使用Spring Quartz实现
 - 6.支持用户头像和产品图上传(存储至阿里云OSS)
 - (本来想引入ElasticSearch结果没有好用的ORM框架而停止)
